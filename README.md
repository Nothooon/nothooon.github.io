# Ouais le blog ouais

## Prérequis

- ruby > 3.0.0
- gem > 3.2.3 

## Installation

Suivre les instruction de la [doc Jekyll](https://jekyllrb.com/docs/installation/) pour installer l'environnement de base.
Ensuite, il faut suivre le [guide d'installation de Chirpy](https://chirpy.cotes.page/posts/getting-started/) pour installer le thème en lui même

L'installation est techniquement optionnelle puisqu'il est tout à fait possible de juste éditer les fichiers markdown sans faire tourner quoique ce soit en local.

### Metadatas

Exemple des propriétés utilisées sur les articles:

```md
---
layout: post
title:  "Final Fantasy I - Des Débuts Humbles"
tags: final-fantasy
category: "Final Fantasy Marathon"
date: "2023-10-15"
lang: fr
image:
  path: /assets/img/posts_preview_images/ff1_logo.jpeg
  alt: Final Fantasy I Logo
```
---
